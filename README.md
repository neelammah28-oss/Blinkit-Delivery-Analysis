# Blinkit Delivery Delay Analysis - Bangalore

### Business Problem
Analyzed Blinkit delivery data to identify operational bottlenecks causing delays in Bangalore.

### Tools Used
SQL, Power BI, Excel and python

### Key Insights
1. *City-wide avg delay: 8.3 mins*, but Whitefield & HSR Layout hit 14+ mins
2. *Peak delay hours: 10am & 7-9pm* - likely office & dinner rush
3. *Recommendation*: Add 15% more riders to Whitefield/HSR during 7-9pm slot
